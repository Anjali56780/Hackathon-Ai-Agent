#  AI Customer Support Agent

##  Project Overview
This project is an AI agent that answers customer queries using ticket data.

##  Tech Stack
- Python
- LangGraph
- LangChain
- Google Gemini API

##  How to Run

1. Open Google Colab
2. Install libraries:
   pip install langgraph langchain langchain-google-genai

3. Add your API key

4. Run the agent:
   python main.py

##  Features
- Classifies user queries
- Routes to correct module
- Gives smart responses

## Sample Queries
- Show ticket summary
- How many tickets are open?
- List pending issues
